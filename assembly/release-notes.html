Release Notes for Vaadin Spreadsheet ${project.version}

Overview of Vaadin Spreadsheet

Vaadin Spreadsheet is an add-on for Vaadin Framework which provides Excel spreadsheet editor functionality for any Vaadin application.

List of known issues is available at the Vaadin Trac

Package Contents

The package contains the following files and directories:

/
 The add-on itself along with it's packaged sources and javadocs, licensing information, README and release notes. 
api/
 Extracted, browsable JavaDocs. 
lib/
 Third party libraries needed by Vaadin Spreadsheet. 
testbench-api/
 API for creating Vaadin TestBench tests for Spreadsheet.

Limitations

Vaadin Spreadsheet ${project.version} has the following limitations:

  • There is no direct support for charts. You can, however, bind data in a spreadsheet to a chart using Vaadin Charts. See the demo for an example.
  • There are no provided toolbars or menus for formatting cells.
  • Vaadin Spreadsheet is heavily based on Apache POI and, as such, inherits all limitations of this library. See Apache POI - HSSF and XSSF Limitations for a partial list of these.
  • There is only limited support for the older XLS format. Please use XLSX for a better experience.
  • As Apache POI is used on the server-side, you should be aware that loading XLSX files will eat memory for each session where loaded. Normal cases are no problem, but make sure to take this into account if you allow users to upload Excel files.
  • Rendering performance is not too great on Mozilla Firefox.

Third Party Dependencies

All dependencies are included in the package in the lib/ directory. Vaadin Spreadsheet directly depends on the following third-party libraries:

  • Apache POI
  • dom4j
  • Apache XMLBeans
  • StAX
  • Apache XML API
  • Apache Commons Codec

Supported Platforms and Browsers

Vaadin Spreadsheet supports the following browsers:

  • Latest Google Chrome
  • Internet Explorer 9, 10, 11
  • Latest Mozilla Firefox
  • Latest Opera
  • Touch devices (read-only)
    • Android 4.3+
    • iOS 8+
    • Windows Surface (no touch support, only mouse)
